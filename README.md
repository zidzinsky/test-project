## Requirements
* Node 10 and later
* npm 7

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```

## ENV Variables
```
PORT=3000
CURRENT_VERSION=v1
ENVIRONMENT=dev
GITHUB_API_URL=https://developer.github.com/v3/
VERSION_CONTROL_PROVIDER=GitHub
```
